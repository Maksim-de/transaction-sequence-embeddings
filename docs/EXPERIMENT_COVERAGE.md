# Ревизия покрытия экспериментов

Этот файл сопоставляет эксперименты из ВКР с тем, как они представлены в текущем репозитории.

## Таблица соответствия


| Эксперимент из ВКР                                    | Где отражен                                                                                                                                       | Комментарий                                                                                                                  |
| ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Базовая иерархическая модель с числовой веткой        | `notebooks/core/TabBert_experiments.ipynb`                                                                                                        | Основной ноутбук с базовой конфигурацией и ключевыми частями пайплайна.                                                      |
| Конкатенация числового признака перед классификатором | `notebooks/variants/TabBert_numeric_concat.ipynb`                                                                                                 | Выделен в отдельный ноутбук и явно описан в начале файла.                                                                    |
| Дискретизация числового признака на бины              | `notebooks/core/TabBert_experiments.ipynb`                                                                                                        | Присутствует как отдельная ветка эксперимента внутри основного ноутбука.                                                     |
| CatBoost-бейзлайн                                     | `notebooks/baselines/catboost_fraud_detection.ipynb`                                                                                              | Отдельный, хорошо изолированный бейзлайн.                                                                                    |
| Окно истории длины 5                                  | `notebooks/core/TabBert_experiments.ipynb`                                                                                                        | В коде и именах чекпоинтов виден соответствующий запуск.                                                                     |

| Сравнение иерархической и альтернативной архитектуры  | `notebooks/core/TabBert_experiments.ipynb`, `notebooks/variants/Tabbert_without_history.ipynb`                                                    | Есть код и альтернативный вариант архитектуры, но часть интерпретации требует чтения ноутбука, а не только названия файла.   |
